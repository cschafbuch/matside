# MatSide — Wrestling Intelligence

Real-time college wrestling rankings powered by The Consensus.

## Local Development

```bash
npm install
npm run dev
```

## Deploy to Vercel

This project is configured for one-click Vercel deployment:

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import this repo
3. Vercel auto-detects Vite + React — just click Deploy
4. Connect your custom domain in Project Settings → Domains
